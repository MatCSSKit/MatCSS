# Responsive

MatCSS is a responsive CSS framework. It is designed to be responsive on all devices. It is also designed to be easy to use and easy to customize.

## Breakpoints

MatCSS uses the following breakpoints:

- `xs`: 0px
- `sm`: 576px
- `md`: 768px
- `lg`: 992px
- `xl`: 1200px
- `xxl`: 1400px
