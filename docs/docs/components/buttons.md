# Buttons

Buttons are an important part of any application. They are used for various purposes, to submit or reset a form, navigate, performing interactive actions such as showing or hiding something in an application on click of the button, etc. Buttons are used to drive CTA (Call to Action) on most pages, so itâ€™s important to make them stand out and inviting to users.
